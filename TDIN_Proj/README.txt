Deployment: 
	Compile the solution, 
	Run Server first,
	Run DinningRoom, 
	Run KitchenBar 
Use Instructions:
	Add an order to a table in the dining room, 
	go to KitchenBar terminal and choose one option, check if the order is there, 
	change the status of the order, 
	Pay table when all orders of that table are done.
